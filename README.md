# Export data from excel to word
Formatting data (templates) from excel .xlsx, export to a .docx (word) file.

### How to run it:
- clone this repository
- execute the folder with python:
    ``` bash
    python excelToWord
    ```
## REQUIREMENTS:
modules :
* pandas, 
* python-docx, 
* xlrd


_pip install pandas_

_pip install python-docx_

_pip install xlrd_
